## This is a Daily Task Tracker/To-Do List application that uses React with JSON Server backend. 

### Includes
 - Authentication
 -	Dark/Light Theme Toggling
 -	Search Bar for finding tasks
 -	Activity Log to track tasks
 -	Dashboard to Track Task Statistics

### Tech Stack:
- React 18+
- React Router v6+
- Axios (for HTTP requests)
-	JSON Server (for backend)
- CSS Modules

### To run the json server on port 5000 use: json-server --watch db.json --port 5000
